# SoupFold vs ViennaRNA Benchmark

This directory contains a benchmarking suite designed to compare the performance, Minimum Free Energy (MFE), and structural predictions of **SoupFold** (`strand_soup.exe`) against **ViennaRNA's** `RNAmultifold` on heterogeneous RNA multi-strand complexes.

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/LeatherBrawler/soupfold-vienna-benchmark.git
   cd soupfold-vienna-benchmark
   ```
2. **Install Python dependencies:**
   ```bash
   pip install pandas numpy jupyter
   ```
3. **External Dependencies:** Make sure ViennaRNA package (`RNAmultifold`, `RNAplot`), Ghostscript (`ps2pdf`), and `strand_soup.exe` are installed and available in your system's PATH.

## Directory Structure

- `Scripts/`: Contains the main benchmarking python script and utility scripts.
- `Notebook/`: Contains Jupyter notebooks for post-processing, analysis, and data visualization.
- `Plots/`: Output directory for structure visualizations.
  - `eps/`: Stores the raw `.eps` plots generated by `RNAplot`.
  - `pdf/`: Stores the converted `.pdf` structure visualizations.
- `Results/`: Stores the generated CSV files and analysis outputs.
  - `Output/`: Stores log outputs and runtime execution logs such as `error.txt`.

## Workflow

The workflow is divided into three main steps: 1) Test case generation and execution, 2) Analysis and visualization, and 3) Format conversion.

### 1. Generate Test Cases & Run Benchmark
**Script:** `Scripts/hetero-comparison.py`

This script is the entry point of the pipeline. It programmatically generates test cases using different RNA triplet sequences (e.g., CUG, CAG, GAU) with varying lengths and strand counts. 
- It executes both SoupFold and `RNAmultifold`.
- It evaluates and categorizes the predicted structures (e.g., identical, identical with shifts/inversions, different base pair counts).
- **Output:** It records runtime, MFE, sequences, and dot-bracket structures into a CSV file (e.g., `heterogeneous_benchmark_results.csv`).

**Usage:**
Run this script from the **root directory** (where `strand_soup.exe` is located) so the results save to the `Results/` folder correctly:
```bash
python Scripts/hetero-comparison.py
```

### 2. Analyze Results & Extract Structures
**Notebook:** `Notebook/benchmark_analysis.ipynb`

Once the benchmarking script has finished, use this Jupyter Notebook to dive deeper into the results.
- It parses the CSV output from step 1.
- It categorizes instances where MFE or predicted shapes differ.
- It filters specific cases of interest and organizes them into new CSV files within the `Results/` directory. 
- It generates `.fold` representation files and uses ViennaRNA's `RNAplot` to automatically render `.eps` visualizations of the structures into the `Plots/` directory.

### 3. Convert Visualizations
**Script:** `Scripts/pdf_converter.sh`

`RNAplot` produces structural graphs in `.eps` format. For easier viewing and compilation, this shell script batch-converts all `.eps` files into `.pdf` documents using `ps2pdf`.

**Usage:**
Run this from inside the **`Scripts`** folder:
```bash
cd Scripts
chmod u+x pdf_converter.sh
./pdf_converter.sh
```

## Dependencies
- **ViennaRNA Package:** `RNAmultifold` and `RNAplot` must be installed and accessible in the system path.
- **SoupFold:** Ensure the `strand_soup.exe` binary is correctly compiled and positioned as referenced in the python script.
- **Python 3.x:** `pandas`, `numpy`
- **Ghostscript:** `ps2pdf` is required for the EPS to PDF conversion script.
